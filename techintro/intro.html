Introduction An overview of the technical details of Devilry ----

Devilry is an open source system for handing in assignment deliveries, and for correcting and providing feedback on the assignments. Devilry uses djangom an open source web application framwork written in Python. Devilry has been planned from the beginning to be a general system for use at educational institutions, supporting all kinds of electronic deliveries.

Devilry has been designed for three groups of users; students, correctors/examiners and administrators. The student interface has been designed to be compatible with most browser, even IE6. The examiner and admin interfaces are designed with more modern web techonolgies requiring modern web browsers to work as expected.
