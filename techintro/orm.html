Object-relational mapping Introduction to the Django ORM ----

Object-relational mapping (ORM, O/RM, and O/R mapping) in computer software is a programming technique for converting data between incompatible type systems in object-oriented programming languages. Wikipedia

Python classes are used to define data models that the Django ORM converts to relational datadbase models. The Django query langauge hides the SQL queries from the programmer, making a database switch very easy(). The Django query language is database independent, making a database switch very easy. Supported databases are PostgreSQL, MySQL, SQLite, Oracle.

ORM has some overhead compared to native SQL queries, but there are many optimization techniques avaialable in Django. Where maximum efficieny is needed, it's even possible to write RAW SQL statements. (More on optimizing database access in Django.
