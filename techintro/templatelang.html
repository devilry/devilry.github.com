Templating Template languages... Why? ----

The Django template language makes creating HTML code a breeze.

Django's template language is designed to strike a balance between power and ease. It's designed to feel comfortable to those used to working with HTML. https://docs.djangoproject.com/en/dev/topics/templates

If you have a background in programming, or if you're used to languages like PHP which mix programming code directly into HTML, you'll want to bear in mind that the Django template system is not simply Python embedded into HTML. This is by design: the template system is meant to express presentation, not program logic. https://docs.djangoproject.com/en/dev/topics/templates
