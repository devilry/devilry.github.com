devilry.core.utils — Devilry core utilities¶

class devilry.core.utils.OrderedDict(*args, **kwds)¶

Bases: dict, UserDict.DictMixin

OrderedDict implementation which works just like OrderedDict in python >2.7. See: http://docs.python.org/dev/library/collections.html#collections.OrderedDict
