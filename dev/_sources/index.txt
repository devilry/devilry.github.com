About devilry
=============

Devilry is a system for handing in electronic deliveries. The system is
designed to be flexible and customizable so that many different needs can be
met. Devilry is made to work with all types of electronic deliveries.

Commiting bugs and issues
############################

Devilry is currently under active development. If you find any bugs or issues, or have ideas for improvements or new features, please report them
at our `github issue tracker`_.

Contributing to Devilry
##########################

Devilry is open source, `BSD license`_. If you would like to contribute to Devilry, please check out our 
`shared branch`_ at github. For documentation on how to contribute, see :ref:`developer-contribute`.
