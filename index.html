Devilry

Open source general purpose delivery system
  • Devilry is easy for students
  • Devilry provides examiners with instant access to what they usually need via their dashboard
  • Devilry has an active open source community, where anyone is welcome.
  • Devilry provides examiners with a detailed overview, enabling them to keep track of their assignment students or groups of students.
  • Devilry comes with powerful tools for every step of the process of producing feedback to students.
  • Devilry can show a detailed overview of all students on a subject within a given period.
  • Devilry allows students to see their own statistics, and administrators to view statistics for each student.
  • Devilry allows administrators to manage each assignment in detail, however they are also provided with shortcuts for common actions.

About

Devilry is a system for handling electronic deliveries. The system is designed to be flexible and customizable so that many different needs can be met. Devilry is designed to work with all types of electronic deliveries.

Get involved

The easiest way of contributing to Devilry is by submitting bugs and issues to our issue tracker. Furthermore, you can suggest improvements and new features through the same issue tracker. If you have the skills, you can even help us develop Devilry.

Documentation

Most of the documentation is available through the wiki Wiki. Developers may want to jump directly to the API docs, however new developers should check out the Wiki first.
