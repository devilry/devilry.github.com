Devilry

Open source general purpose delivery system
  • Devilry handles all parts of the electronic delivery and feedback process.
  • Devilry is is a complete tool for course administrators that want to have full control over their assignments, students and examiners.
  • Devilry is easy and effective for course administrators with simple needs.
  • Devilry provides the information course administrators need about their students.
  • Devilry has powerful tools for course administrators with lots of students.
  • Devilry is open source software, and it is based on several open source libraries and tools
  • Devilry is easy for students.
  • Devilry has excellent support for cooperative project assignments.
  • Devilry provides examiners with a TODO-list and a detailed overview, enabling them to keep track of their students.
  • Devilry has an active open source community where anyone is welcome
  • Devilry comes with powerful tools for every step of the process of producing feedback to students

About

Devilry is a system for handling electronic deliveries. The system is designed to be flexible and customizable so that many different needs can be met. Devilry is designed to work with all types of electronic deliveries.

Source code and license

The Devilry source code is available at our repository under the BSD license, which is an open source license.
